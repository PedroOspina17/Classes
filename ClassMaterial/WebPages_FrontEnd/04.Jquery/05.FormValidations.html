Validations on forms

Most of the controls in a form need to be validated to make sure the value typed is correct. There are several validations that can be applied on forms but the most common ones are the following:

  • Required: This validation is to verify that the user have typed something in the input.
    To check this just take the input's value and compare it with an empty string. Remember to use the function trim to erase all the white spaces.$("#fieldToCheck").val().trim() == ""
  • Length: This validation is to verify that the value does not exceed an specific length. $("#fieldToValidate").val().length
  • Dates: When you are working with dates you need to convert the values into a Date data type. To do so you need to call the new Date(). To get the current date you just call the new Date() without parameters.
    Then you can compare dates between them. For example to validate if the field dobDate is less than the current date you you use: new Date($("#dobDate").val() + " 00:00:00") < new Date() .
