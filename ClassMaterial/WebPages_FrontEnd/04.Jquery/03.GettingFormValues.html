Information manipulation.

There are some specific methods that are used to extract data from the tags. Each of the three following functions can be used with and without parameters. If there is not parameters it gets the value of the element, if there is a parameter it changes the value of the element.

.val(): is often used to get the value of the controls.
  • On inputs: Is the value written on the text box, no matters what type it is.
  • On Selects: it is the value of the option selected.
            I.e: 
            - To get the value of the input 'User'
                $("#user").val();
            - To change the value of the input 'User'
                $("#user").val("This is the new value of the input");
.text(): it is used to get the text inside the tags. 

            I.e: 
            To get the content of the h2 'title': 
                $("#title").text();
            To change the content of the h2 'title': 
                $("#title").text("The title was change with JQuery");
.attr("attirbuteName"): It is used to get any attribute value in a element. You should specify the attribute you want to get in the parameters. 
            I.e: 
            - To get the 'placeholder' attribute of the input 'User'
                $("#user").attr("placeholder");
            - To change the 'placeholder' attribute of the input 'User'
                $("#user").attr("placeholder", "This is the new placeholder");
