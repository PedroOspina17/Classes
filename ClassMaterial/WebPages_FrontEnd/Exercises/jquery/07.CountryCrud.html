Form CRUD

A crud is an acronym of Create. Read. Update. Delete.

This is the very beggining of forms. Putting togeter all the knowledge, create a crud for countries.

In this exercise we are going to start with the create and the read. Please follow the instructions bellow:

  1. Create a form with three fields:
    • Country name:A text field, required and 15 characters long max.
    • Long country name:A text field, non required and 50 characters long max.
    • Country description:A text field, non required but 10 characters min if something is typed and 250 characters long max.
  2. Create a button called "Cancel" to clean all the form.
  3. Create a button called "Save" to do the following:
    1. Validate all the restrictions descripted in the item #1.
    2. If any validation is not correct show an alert describing all the not met validations.
    3. If all the validations are correct, the information written in the form should be displayed inside a table. Then clean the form to allow the user to continue typing information.
