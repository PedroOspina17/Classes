DOM Manipulation

The dom (Document object model - Page structure) can be modified on the fly. This means that using javascript or Jquery we can add, modify or delete elements as needed. This can be done with the following methods:

  1. .append(ElementToAdd): This method is used to add a new element inside and at the end of an specific tag. Structure: $("#ElementToModify").append(ElementToAdd) The element to add can be a variable or the html directly. 
                            Example: Add an input button to my div with ID 'divToModify'
                            $("#divToModify").append("<input type="button" value="Test button">");
                        
                        Similar to the append there is the prepend(ElementToAdd) which also adds a element inside another one but at the beggining. 
                        This is used when you already have some content but you want your new element at the top of the list

  2. .remove(ElementToRemove): This method is used to eliminate a element from the page. Structure: $("#ElementToRemove").remove() 
                        Example: Select the input with id 'userId' and delete it from the page.
                    $("#userId").remove()
  3. .empty(): This method is used to remove all the elements that are inside an opening tag and a closing tag. Structure $("#IdElementToChange").empty() 
                         Example: Remove all the elements inside the div with id 'divToClean' which has the following structure:
                        <div id="divToClean">
                        
                            <input type="text" id="input1" name="input1" />
                            <input type="text" id="input2" name="input2" />
                            <input type="text" id="input3" name="input3" />
                            <input type="text" id="input4" name="input4" />
                        
                        </div>

                        $("#divToClean").empty()

                        After the command below is executed the result is:

                        <div id="divToClean">
                        
                        </div>
