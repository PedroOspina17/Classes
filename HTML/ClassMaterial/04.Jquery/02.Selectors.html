Selectors

jQuery selectors allow you to select and manipulate HTML element(s). jQuery selectors are used to "find" (or select) HTML elements based on their name, id, classes, types, attributes, values of attributes and much more.

  • By tag name

    Selects all the tags elements specified.

     $("TAGNAME")
    Example: selector to get all the input tags 
    $("input")
  • By class name

    Selects all the elements with the class specified. Denoted with "."

     $(".CLASSNAME")
    Example: selector to get all the inputs with class 'testClass' 
      $(".testClass")
  • By element id

    Selects the first the element with the id specified.Denoted with "#"

     $("#ElementId")
    Example: selector to get the element with id 'user'
      $("#user")
  • By property

    Selects all the elements with the property specified. Denoted with "[]" 

     $("[Property]")
    Example: selector to get all the elements with the property 'href'
      $("#user")
  • By property and value

    Selects all the elements with the property and value specified. Denoted with "[]"

     $("[Property='Value']")
    Example: selector to get all the elements with the property 'type' and value 'number'
      $("[type='number']")
     This can be mixed with the tag element selector. Example: selector to get all the inputs elements with the property 'name' and value 'dobDate'
      $("input[name='dobDate']")
