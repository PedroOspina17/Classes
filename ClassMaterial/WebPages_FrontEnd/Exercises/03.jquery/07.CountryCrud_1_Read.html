Form CRUD

A crud is an acronym of Create. Read. Update. Delete.

This is the very beggining of forms. Putting togeter all the knowledge, create a crud for countries.

Read operation:

This is the easiest operation. The purpose is to read all the information that is already registered and show it in the form.

  1. (HTML, css, bootstrap) Create a table with three columns: Name, Description and status. This step is just to create the table and the header columns (the structure only, the information goes later)
    Hint: Remember to assign a table's Id to be able to fill it with the information in next steps
  2. (Javascript variables) Create a global javascript variable to store a list of json information. The variable should have three rows with the following fake information (Id, Name, LongName, Description, PopulationAprox, Status): 
    1. 2201   |   Colombia        |  Colombia's long Name        |   South-american country  |   15.000      |   Active
    2. 2202   |   Brasil          |  Brasil's long Name          |   South-american country  |   150.000     |   Inactive
    3. 2203   |   Peru            |  Peru's long Name            |   South-american country  |   5.000       |   Active
    4. 2204   |   EEUU            |  EEUU's long Name            |   North-american country  |   1.500.000   |   Active
    5. 2205   |   Argentina       |  Argentina's long Name       |   South-american country  |   500.000     |   Active
    Remember: To create a json list variable in javascript follow this notation: 
                            var VariableName = [
                            {
                                "fieldName" : value,

  3. (Javascript events, on load functions) When the page is loaded, iterate the variable created above and take every row, one by one. (Try to check this by printing in the console each element while you are iterating and some useful messages such as the country's name or PopulationAprox to verify that you have access to the entire information.)
    Hint: To iterate a list use the jquery's foreach. $.each(ListToIterate, function(index, value) { // All the code Goes here !!! }
    Hint 2: To access to an specific field of the json use the dot. Example: To access to the field "Name" in a json use: elementList.Name
  4. (Javascript loops, DOM modifications) Having the information one by one (Inside the loop) add a new row in the table (DOM modification) following the algorithm described bellow:
    1. Create the HTML for the row to be inserted in the table.
      Hint: Assign an specific and unique id for each row for next operations to be easy to identify among others.
      Example: 
                                  < table>
                                < tr id="Row_IDENTIFIER"> < /tr>

                                < tr id="Row_1520"> < /tr>
                                < tr id="Row_2250"> < /tr>

    2. Create a jquery selector to get the table created in step #1. (Remember to use the id assigned.)
    3. Insert the html created in the first step into the table selected in the previous step
