Form CRUD

Create operation:

In this exercise we are going to start with the create opeation by creating the form and the validations. The algorithm is described bellow, please follow it carefully.

  1. (HTML, CSS, bootstrap) Create a form with these fields:
    • Id:A numeric text field, should be a positive unique number and required.
    • Country name:A text field, required and 15 characters long max.
    • Long country name:A text field, non required and 50 characters long max.
    • Country description:A text field, non required but 10 characters min if something is typed and 250 characters long max.
    • Population aprox:A numeric field to specify how many habitants are in the country approximatelly
    • Status: A boolean field to specify if the country is active or not. Active will be the default status.
  2. (Jquery, Events and functions, Change input's value) Create a button called "Cancel" to clean all the form. (Set all the input values to blank)
  3. (Get input's value, Validations, Sweetalerts, DOM modifications) Create a button called "Save" to do the following:
    1. Validate all the restrictions described in the item #1.
    2. If any validation is not correct show an alert describing all the not met validations.
    3. If all the validations are correct, the information written in the form should be displayed inside a table. (The way you did it in the read operation)
    4. Clean the form to allow the user to continue typing new information. (The way you did it in the cancel button)
    5. Show a toastr informing the result of the operation.
