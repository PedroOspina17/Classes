DOM Manipulation

The dom (Document object model - Page structure) can be modified on the fly. This means that using javascript or Jquery we can add, modify or delete elements as needed. This can be done with the following methods:

  1. .append(ElementToAdd): This method is used to add a new element inside and at the end of an specific tag. Structure: $("#ElementToModify").append(ElementToAdd) The element to add can be a variable or the html directly. 
                            Example: Add an input button to my div with ID 'divToModify'
                            $("#divToModify").append("<input type="button" value="Test button">");
                        
                        Similar to the append there is the prepend(ElementToAdd) which also adds a element inside another one but at the beggining. 
                        This is used when you already have some content but you want your new element at the top of the list

  2. .remove(): This method is used to eliminate a element from the page. Structure: $("#ElementToRemove").remove() 
                        Example: Select the input with id 'userId' and delete it from the page.
                    $("#userId").remove()
  3. .empty(): This method is used to remove all the elements that are inside an opening tag and a closing tag. Structure $("#IdElementToChange").empty() 
                         Example: Remove all the elements inside the div with id 'divToClean' which has the following structure:
                        <div id="divToClean">
                        
                            <input type="text" id="input1" name="input1" />
                            <input type="text" id="input2" name="input2" />
                            <input type="text" id="input3" name="input3" />
                            <input type="text" id="input4" name="input4" />
                        
                        </div>

                        $("#divToClean").empty()

                        After the command below is executed the result is:

                        <div id="divToClean">
                        
                        </div>
  4. $("ElementToCreate"): There are several ways to achieve the same goal. You can create elements with the following structure $("TagToCreate") and then add it to the dom with the methods described bellow. This is specially used when you need to create nested elements. 
                        Start creating the elements as a container. It  can be any tag element
                    
                        var container = $("<someHtmlTag anyProperty="AnyValue"></someHtmlTag>")
                    

                    Then, create all the elements inside of the container and add them using the append. 
                    
                        container.append("SomeOtherElement");
                        container.append("SomeOtherElement");
                        container.append("SomeOtherElement");
                        container.append("SomeOtherElement");
                    

                    And when all the element is created, it needs to be added to the DOM 
                    using the append but this time modifying the element from the html and adding the container variable.
                    
                        $("#ElementToModify").append(container);
                    
                
                        Example: 
                    Create a div with a label and a input inside (A bootstrap form field) each time the button is clicked

                     Before:
                    <div id="divToModify">
                    </div>

                    Script:
                    var container = $("<div class="form-group"></div>")
                    container.append("<Label for="input" > Some text label </Label>")
                    container.append("<input type="text" id="input" name="input2" />")
                    $("#divToModify").append(container);

                    <div id="divToModify">
                        <div class="form-group">
                            <Label for="input" > Some text label </Label>
                            <input type="text" id="input" name="input2" />
                        </div>
                    </div>
    Need to notice that $("div") is different than $("<div>Content</div>")
    1. $("div") is used to select all the 'div' tag elements
    2. $("<div>Content</div>"): is used to create a 'div' element in memory

TOASTR

Toastr is another library to show messages to the user. Similar to the sweet alert it needs to be imported in the same way of the swal

Including the script
<script src="../Common/js/sweetalert.min.js"></script>
and the styles:
<link rel="stylesheet" href="../Common/css/toastr.min.css">

There are 4 type of message you can display. It just change the style of the message shown

  • success: green
  • error: red
  • warning: orange
  • info: blue

To use the toastr you just need to follow this structure toastr.<<typeOfMessage>>('Any message you want.')

                Example:
                * To show a info message saying "Hello world". Use:

                
                    toastr.info("Hello world").
                

                * To show a warning message saying "The form has invalid values.". Use: 
                
                    toastr.warning("The form has invalid values.");
