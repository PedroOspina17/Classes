Web storage

In this exercise you will be apply the knowledge about local storage to create a simple application that persist the information saved.

Please read and follow the requirements described bellow:

  1. Create a form with the following fields:
    • Key: A Text field, Required, Unique, 3 characters min and 10 max. Should be only one word
    • Value: A text field, Required and 250 character max.
  2. Create a button called "Create data" which needs to:
    • collect the fields created in the step #1
    • validate them
    • if all the validations are met, create a definition list with the information collected.
  3. Inside every definition term create a button to allow the deletion of the item.
  4. Create a button called "Clean data" to allow the user to delete all the entries.
  5. When the page is loaded the user should be able to see the information that was created before
